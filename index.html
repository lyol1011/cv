Crypto Market Analysis

Problem

Data

Historische Marktdaten über CoinGecko API sowie Bitcoin-On-Chain-Daten über Blockstream API.

Approach
  • Explorative Datenanalyse und Visualisierung
  • Volatilitäts-/Volumenmuster nach Wochentag und Tageszeit
  • Marktkapitalisierungsvergleiche
  • Experimentelle Event-Kontextualisierung via OpenAI API
Outcome

Kombination aus strukturierter Marktdatenanalyse und experimenteller LLM-Integration zur Kontextualisierung starker Marktbewegungen.

Occupancy Detection (ML)

Problem

Vorhersage der Raumbelegung auf Basis von Sensordaten, um HVAC-Systeme bedarfsorientiert zu steuern.

Data

Zeitreihen-Sensordaten mit Temperatur, Luftfeuchtigkeit, CO2 und weiteren Merkmalen.

Approach
  • EDA und Datenbereinigung
  • Feature Engineering (Zeitmerkmale, nichtlineare Features, kurzfristige Dynamik)
  • Modelltraining (Logistic Regression, Decision Tree, Random Forest)
  • GridSearchCV + TimeSeriesSplit
  • Threshold-Tuning unter Kosten-/Komfortkriterien
Outcome

Random Forest zeigte im Projekt das beste Verhältnis zwischen Komfort und Betriebskosten.
